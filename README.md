# ✨ Social Media Caption Generator

AI-powered caption generator built with Python, Streamlit and Groq AI.

## Features
- Multiple platforms (Instagram, Twitter, LinkedIn)
- Multiple tones (Funny, Inspirational, Professional)
- Generate 1-5 captions at once
- Free to use (Groq API)

## How to Run
pip install -r requirements.txt
streamlit run app.py

## Tech Stack
- Python
- Streamlit
- Groq AI (LLaMA 3.3)
